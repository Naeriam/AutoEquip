# Auto Equip (Addon for World of Warcraft) [![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Auto equip gear with higher ilvl
Perfect for alternative characters and events such as Remix, where you go without looking, equipping yourself with whatever has the highest ilvl without caring.

You can download this addon from: https://www.curseforge.com/wow/addons/auto-equip

## Considerations
- It never equips weapons
- Only equips **soulbound** items
- It respects your class armor type
- Disabled during combat
- Disabled while the bank is open
